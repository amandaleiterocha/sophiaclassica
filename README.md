Qual é a ideia central de cada obra? Obter a sinopse concisa é o que permite identificar, de forma direta e inequívoca, o título ideal para aprofundamento. 
Essa é a função primordial do "SOPHIA CLÁSSICA: Títulos, Autores e Sinopses Concisas".

MODO DE USO: Para iniciar, utilize a barra de busca inserindo o Título, o Autor ou uma Palavra-Chave filosófica relevante. 
O site fornecerá uma sinopse concisa. Caso necessite de dados complementares sobre a obra, clique no botão "Saiba Mais".
